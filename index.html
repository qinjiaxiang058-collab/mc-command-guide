常用参数速记

<targets>
玩家或实体选择器，如 @p、@a、@e[type=zombie]。
<pos>
坐标；支持绝对坐标、相对坐标 ~、局部坐标 ^。
<nbt>
Java NBT 数据；常用于 /data、/summon、物品组件。
<resource>
命名空间 ID，如 minecraft:stone、custom:boss。

读语法

<必填> 必须输入，[可选] 可以省略，竖线表示多选一。

语法与执行环境

聊天栏输入命令需要前置 /；服务器控制台通常不需要。命令方块、函数文件和 run_command 文本事件也能执行命令。多数管理命令要求开启作弊或拥有足够 OP 权限。

选择器

@p 最近玩家，@a 所有玩家，@r 随机玩家，@e 所有实体，@s 执行者。

权限级别

0 普通可用，2 常用作弊/命令方块，3 管理玩家，4 控制服务器进程或性能采样。

命名空间

未写命名空间时通常默认 minecraft:。数据包命令建议显式写出命名空间。

返回值

很多命令可配合 /execute store 存储结果，适合计分板、数据包和自动化逻辑。
